1.Создать программу, содержащую классы точка (Point), ребро (Edge), задаваемое двумя точками и треугольник (Triangle), задаваемый тремя точками. 
2.Реализовать функционал определяющий периметр (Perimeter) и площадь (Area) треугольника. 
3.Реализовать функционал, позволяющий определить является ли треугольник прямоугольным (Right) и равнобедренным (Isosceles). 
4.Составить массив треугольников и написать код, вычисляющий средний периметр всех прямоугольных треугольников и среднюю площадь всех равнобедренных треугольников.

Требования:
1.Грамотная объектная декомпозиция и абстрагирование - разделение на классы, выделение полей и методов классов;
2.Обработка некорректных данные при создании объекта.
3.Использование конструкторов для создания объектов;
4.Грамотная декомпозиция алгоритм на классы и методы;
5.Корректное разбиение проекта на файлы;
6.Соблюдение соглашений по именованию и форматированию кода: корректное именование модулей с кодом и заголовочных файлов, классов, методов и полей классов, переменных.

Бонусные задания:
1.Реализация алгоритма для n-угольников.
